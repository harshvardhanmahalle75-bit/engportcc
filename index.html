Project Overview

A detailed creep and fatigue assessment for HRSG Unit 3 at Immingham Power Station. Simulating thermal and pressure loading over steady and cyclic scenarios using FEA.

Scope of Work

Reconstructed CAD for:

  • HP Superheater 1: 7 headers + 1 manifold
  • HP Superheater 2: 3 headers + 2 manifolds
  • Attemperator: 1 Unit
Key Findings

Creep Hotspots

  • UH1 (HP SH 2): 17.87% usage
  • LH3 (HP SH 1): 16.49% usage

Fatigue Critical

  • UH1 (HP SH 1): 18.03% usage
Recommendations & Value
  • Implement real-time thermal instrumentation on key headers
  • Target NDT at nozzle and branch intersections
  • Reduced uncertainty for operation through 2047

ASME Section I P91 Steel VPI Immingham
▾ View Detailed Report
Replacement Design
Problem Statement

Replacement of a socket weld fitting—prohibited in high-pressure steam service—with a full penetration welded set-on connection.

Design Approach

Three sensitivity cases evaluated for robustness:

  • Case 1 (Baseline): Real geometry assumptions.
  • Case 2 (Material): P91 material representation.
  • Case 3 (Geometry): Bounding reduced thickness case.
Key Results

Usage Ratios

  • Main Body: 31% – 39% (Safe)
  • Nozzle: 39% – 68% (Safe)

Justification

  • Full penetration weld for fatigue resistance
  • P91/F91 metallurgical compatibility
Value Delivered
  • Eliminated socket weld failure risk
  • Full compliance with VPI site engineering standards
  • Reduced inspection burden
